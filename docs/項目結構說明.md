## 🚀 使用說明

### 運行項目
1. 確保所有文件都在正確的目錄結構中
2. 直接在瀏覽器中打開 `index.html`
3. 項目會自動跳轉到登入頁面

### 文件整理（可選）
如果需要重新整理文件結構，可以運行 `整理文件結構.bat` 腳本。

### 開發注意事項
- 所有路徑都使用相對路徑
- 新增資源文件時請放在對應的 `assets/` 子目錄中
- 修改CSS/JS文件後記得更新HTML中的引用路徑
